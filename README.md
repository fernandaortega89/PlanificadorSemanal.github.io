#Estudio Organizado — Planificador Semanal
Es una aplicación web sencilla y visual para organizar tus tareas de estudio durante la semana.  
Desarrollada para funcionar 100% en el navegador, sin cuentas ni base de datos, guardando todo localmente mediante **localStorage**.


#Características principales

- Planificador semanal (Lunes a Domingo)  
- Crear, editar y eliminar tareas fácilmente  
- Arrastrar y soltar tareas entre días  
- Prioridades: alta, media y baja  
- Guardado automático en el navegador  
- Exportar e importar tareas (archivo JSON)  
- Botón para limpiar la semana  
-  Modo offline (funciona sin conexión)  



#Tecnologías utilizadas

- **HTML5****
- **GitHub Pages**
